## 2026-04-10 (v1 → v2)

### The Core Problem 섹션 리뉴얼

**카드 콘텐츠 교체**
- 기존 3개 카드(고비용·일회성 모델 / 데이터 단절과 신뢰 붕괴 / 변화에 무딘 정적 모델) → 4개 카드로 교체
  1. 정적인 역량 모델
  2. 주관적 평가 기준
  3. 느린 업데이트 주기
  4. 제한적 데이터 활용
- 카드 상단 라벨(예: "고비용·저효율 구조") 제거 — 제목과 중복된 내용이었음

**레이아웃 변경**
- 3열 그리드(`md:grid-cols-3`) → 가로 한 줄 4열 반응형 그리드
  - 모바일: 1열 / 태블릿(sm): 2열 / 데스크탑(lg): 4열
- 카드 패딩·폰트 크기 소폭 조정(p-7→p-6, text-xl→text-base) — 4열 배치에서 여유감 확보

### 스크롤 탑버튼 추가

- 페이지 우하단 고정(`fixed bottom-6 right-6 z-50`) — 콘텐츠 가림 없음
- **스크롤 감지 로직**
  - 300px 미만: 버튼 완전 숨김(`opacity-0 + pointer-events-none`)
  - 300px 이상 스크롤 중: 버튼 표시
  - 스크롤 멈춤 후 1.2초 경과: 자동 숨김 (타이머 방식)
- `cursor-pointer` 적용 — 호버 시 포인터 커서 표시
- `translate-y` 트랜지션으로 등장/퇴장 애니메이션 적용
- 모바일 반응형: `w-11 h-11` (모바일) / `w-12 h-12` (sm 이상)
- 클릭 시 `window.scrollTo({ top: 0, behavior: "smooth" })` 부드러운 상단 이동
