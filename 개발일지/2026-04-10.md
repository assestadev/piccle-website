### Core Problem 섹션
- 카드 콘텐츠 교체
  - 기존 3개 카드 → 4개 카드로 변경
  - 기존: 고비용·일회성 모델 / 데이터 단절과 신뢰 붕괴 / 변화에 무딘 정적 모델
  - 변경: 정적인 역량 모델 / 주관적 평가 기준 / 느린 업데이트 주기 / 제한적 데이터 활용
- 카드 상단 라벨 제거
  - 제목과 내용이 중복되어 삭제 (예: "고비용·저효율 구조")
- 레이아웃 변경
  - 3열 그리드 → 가로 한 줄 4열 반응형으로 변경
  - 반응형: 모바일 1열 / 태블릿(sm) 2열 / 데스크탑(lg) 4열
  - 카드 패딩·폰트 소폭 축소 (p-7→p-6, text-xl→text-base) — 4열 배치 여유감 확보

### 스크롤 탑버튼
- 신규 추가
  - 우하단 고정 (`fixed bottom-6 right-6 z-50`) — 콘텐츠 가림 없음
  - 300px 미만 스크롤: 숨김
  - 300px 이상 스크롤 중: 표시
  - 스크롤 멈춤 후 1.2초 경과: 자동 숨김 (타이머 방식)
  - 클릭 시 부드러운 상단 이동 (`behavior: "smooth"`)
  - 호버 시 `cursor-pointer` 및 `translate-y` 애니메이션 적용
  - 반응형: 모바일 44px / sm 이상 48px
