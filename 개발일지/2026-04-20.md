## Use Case 섹션 차트 hover 재생 시도 및 원복 (`app/page.tsx`)

### 시도 내용 (옵션 B — hover 트리거 재생)
- `anim1/2/3` 상태를 `card1/2/3 { key, staged }` 구조로 교체
- 각 차트 카드에 `onMouseEnter` 핸들러 추가
- 마우스 진입 시 차트 `key` 증가(강제 리마운트) → staged 리셋 후 딜레이 재실행으로 재생 구현
- `animating` ref로 중복 호버 방지

### 원복 이유
- 두 번째·세 번째 시리즈(이상적 인재상 레이더 / 도입 후 막대)가 늦게 등장하는 타이밍이 어색함
- Recharts 기반 staged 애니메이션 특성상 hover 재생 UX가 매끄럽지 않아 전 버전으로 되돌림

### 현재 상태
- 스크롤 최초 진입 시 1회 애니메이션 (IntersectionObserver) 유지
